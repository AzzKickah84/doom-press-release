"I wanted one package," said port author Thijs Boekholt, speaking exclusively to UAC Wire between playtest sessions on E1M1. ■ The Arsenal — Everything New & Improved

Classic DOS Startup Console

Launch the game and watch the original text-mode boot sequence: startup banner, V_Init, Z_Init: Init zone memory allocation daemon, R_Init: Init DOOM refresh daemon, and every subsystem line — just like 1993. The console closes when play begins.

NEW IN 3.0.3

DOOM Forge 3.1

Live sector fill from floor textures, 32-step undo/redo, Ctrl+F texture search, thing rotation, upper/mid/lower wall slots, sector & vertex delete, two-sided shared edges, F2 map lump rename, and correct Test-in-DOOM warp.

NEW IN 3.0.3

3.0.2 Quality Pass

Launcher No Monsters / Respawn / Fast options work again; IWAD and PWAD paths persist; renderer Auto applies on launch. Forge thing select, drag, and delete; line delete keeps sidedefs consistent. Bot deathmatch respawn crash fixed; VS CPU menu back-navigation and skill selection corrected; main-menu credit no longer overlaps Quit.

NEW IN 3.0.2

Smarter Bot Combat

Deathmatch bots rush the nearest foe, face them, and open fire — no more endless laps around each other. Aim prediction and wider firing angles make bot duels feel like real player fights.

NEW IN 3.0.1

Bots Use the Whole Map

Co-op and deathmatch bots pick up keys, health, armor, ammo, and weapons they need. They press switches and doors to progress, hunt monsters without line-of-sight, and follow you through the level.

NEW IN 3.0.1

DOOM 3.0 — Major Release

The 2.x feature line graduates to version 3: CPU bots, DOOM Forge, CTF, demo recording, persistent settings, full WAD bundle, mouse-look, and the complete launcher experience ship under one major version number.

NEW IN 3.0.0

Aggressive CPU Bot AI

Deathmatch bots hunt the nearest enemy, chase through the map even without line-of-sight, and fire only when they have a clear shot. Players respawn after death in bot deathmatch — no more stalemates behind cover.

IMPROVED IN 3.0.0

Launcher Layout Fix

Footer buttons no longer overlap Game tab controls. PLAY DOOM, MAP FORGE, CHANGELOG, and ABOUT sit below the tallest tab content — including CPU player options, resolution, and video renderer settings.

FIXED IN 3.0.0

CPU Player Bots

Finally, deathmatch without waiting for dial-up. Add 1–3 computer opponents for offline fragfests on any map with deathmatch spawns, or bring co-op allies that shoot demons and follow you through the campaign. Full player slots with weapons, strafing, and aim — not monster retextures.

NEW IN 2.0.14

VS CPU — Main Menu & Launcher

Title menu: VS CPU PLAYERS → Deathmatch or Co-op → pick 1–3 bots → skill & map. Launcher Game tab: enable CPU players, set mode, count, and skill before you launch. Flags: -bots 2 -deathmatch or -bots 1 -botcoop.

NEW IN 2.0.14

Settings That Stick

Controls, display options, and key bindings save to %LOCALAPPDATA%\DOOM\default.cfg — your personal profile, not the install folder. Reinstall or run from Program Files without losing your layout. Existing configs in the game folder migrate automatically.

NEW IN 2.0.13

Complete WAD Bundle

The installer ships every *.wad from the build folder — IWADs, PWADs, and extras like doom_extra.wad. No more hunting for add-on maps after setup.

NEW IN 2.0.13

AZZ_DOOM Episode

The UAC Territory Breakout episode (AZZ.WAD, AZZ_DOOM.WAD, AZZ_DOOM.TXT) is included in the installer with a Start Menu readme shortcut. Load the PWAD from the launcher and dive in.

NEW IN 2.0.13

Custom Display Resolution

Eight window-size presets from classic 320×200 up to 2560×1600. Set in Options → Display or from the launcher Game tab. The game still renders at authentic 320×200; your chosen size controls how large the window appears on screen.

NEW IN 2.0.12

Texture Filter (Sharp / Smooth)

Choose pixel-perfect Sharp upscaling or Smooth linear filtering when the framebuffer is stretched. Best results with OpenGL or Direct3D presentation. Toggle in Options → Display or the launcher.

NEW IN 2.0.12

Demo Record & Playback

Record your runs from the main menu or launcher — saved as .lmp files in the install folder. Play back demos from a slot list. Press Q to stop recording. Launcher Demos tab adds, removes, and renames demo files before launch.

NEW IN 2.0.11

Mouse-Driven Menus

Point-and-click every in-game menu with a visible arrow cursor. Hover highlights items; left-click activates; right-click goes back. Sliders respond to click position. Keyboard and gamepad still work — keyboard navigation was restored in 2.0.12.

IMPROVED IN 2.0.12

CTF Conversion Wizard

Forge guides you through turning any map into a Capture The Flag arena: flag bases (9010/9011), team spawns, distance checks, and one-click Test CTF. Pulsing map hints show where to click next.

NEW IN 2.0.11

DOOM Forge Map Editor

Draw vertices, lines, and sectors. Paint floor and ceiling textures from a thumbnail browser. Place all 126 mobj types with live sprite previews. 3D fly-through preview with WASD. Save and load PWADs without third-party tools.

BUILT-IN WAD EDITOR

Mouse-Look & Crosshair

Full mouse look: turn with mouse X, pitch with mouse Y (keyboard-only movement). Relative mouse capture for smooth 360° spinning. Safe pitch limits and synced renderer math keep wall textures stable when looking up or down. Optional crosshair in Options → Display.

FIXED IN 2.0.10

Custom Controls

Rebind every action from Options → Controls. One-click WASD or Arrow Key presets. Your layout saves to your per-user default.cfg (see Settings That Stick).

NEW IN 2.0.8

Capture The Flag

Red vs Blue team multiplayer. Grab the enemy flag, capture at your base, first to five wins. Dedicated flag-base things (9010/9011) placeable in Forge. Intermission scoreboard and automatic map restart.

MULTIPLAYER

GTA-Style Radar

Bottom-left minimap with fog-of-war — only explored walls appear, matching automap logic. Brighter player marker. Toggle in Options → Display.

HUD UPGRADE

Mind Control Weapon

Turn demons into allies with the experimental Mind Control device. Enabled from the launcher. Converted monsters fight for you — and in 2.0.12 you can still kill them with normal weapons if you change your mind. Graphics delivered via doom_extra.wad.

FIXED IN 2.0.12

DOOM Launcher

Pick IWAD and PWAD files, skill, episode, map. Network games for up to four players. CPU players for offline bot matches. Demos tab: play, record, add, remove, and rename .lmp files. Display Resolution and Texture Filter on the Game tab. Video renderer: Auto, OpenGL, Direct3D, or Software. Launches Forge with one click. E1M1 music while you configure.

PRE-GAME GUI

Windows & macOS

Full Windows 10+ installer with Start Menu shortcuts and changelog. Native macOS .app bundles with DMG and PKG installers for the complete trilogy: game, launcher, and Forge.

CROSS-PLATFORM

Cheat Code Reference

Every classic cheat documented in CHEAT_CODES.txt — IDDQD, IDKFA, IDCLIP, level warps, message toggles, and automap secrets. Ships in the Start Menu for quick lookup.

Polish & Personality

42 new rude quit messages. Redesigned in-game Options menu. Forge Win2000 classic UI. Sprite previews fixed across 126 types. Custom DoomClon icon on game, launcher, and installer.
